## ⚙️ Configuration

Settings are stored in `config.json` at `%APPDATA%\AmongUsTrainer\config.json`.

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `menu_hotkey` | String | `"Delete"` | Key to toggle menu |
| `auto_attach` | Boolean | `true` | Auto-attach on startup |
| `process_name` | String | `"Among Us.exe"` | Target process |
| `esp_enabled` | Boolean | `true` | Enable ESP overlay |
| `safe_mode` | Boolean | `true` | Restricts unstable features |
| `log_level` | String | `"info"` | `"debug"`, `"info"`, `"error"` |
